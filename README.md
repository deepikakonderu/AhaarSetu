# 🍲 AhaarSetu – Food Wastage Rescue Platform

## 📌 Overview

AhaarSetu is a web-based platform designed to reduce food wastage by connecting food donors with NGOs, shelters, and organizations in need. The platform enables donors to post surplus food and allows receivers to request and collect available food efficiently.

## 🚀 Features

### Donor Module

* Donor Registration & Login
* Post Food Donations
* View Donation History
* Manage Food Availability

### Receiver Module

* Receiver Registration & Login
* Browse Available Food Donations
* Request Food
* Track Request Status

### General Features

* User Authentication
* Responsive User Interface
* Real-Time Food Listing Management
* Dashboard for Donors and Receivers

## ⚙️ Installation

```bash
git clone https://github.com/deepikakonderu/AhaarSetu.git
cd AhaarSetu
npm install
npm run dev
```

## 🔑 Environment Variables

Create a `.env` file in the project root:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```
